# Quiz Leaderboard System

This is a Java program for the internship assignment. It connects to an API, processes quiz scores, handles duplicate data, and submits the final leaderboard.

## Requirements
* Java Development Kit (JDK) 11 or higher
* VS Code with the Java Extension Pack

## How to Run in VS Code
1. Open the folder containing `QuizLeaderboard.java` in VS Code.
2. Open the `QuizLeaderboard.java` file.
3. Click the **Run** button above the `public static void main` method.
4. Wait for about 50 seconds for the program to complete all 10 API calls.
5. The terminal will display the success message.
